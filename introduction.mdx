## What is Bloc de Notas?

Bloc de Notas is a full-stack notepad application that lets you create, organize, and track notes with completion status. It uses a PostgreSQL database with stored procedures to handle all data operations and an HTML frontend for a straightforward user interface.

## How it works

Bloc de Notas is composed of three layers:

1. **PostgreSQL database** — stores notes in the `notes` table and exposes five stored procedures for CRUD operations.
2. **Backend server** — calls the stored procedures in response to HTTP requests from the frontend.
3. **HTML frontend** — a single-page interface where users create and manage their notes.

## Database functions at a glance

| Function | Description |
|---|---|
| `notes_agregar` | Insert a new note |
| `notes_buscar` | Find a note by ID |
| `notes_listado` | List all active notes |
| `notes_modificar` | Update a note's fields |
| `notes_eliminar` | Soft-delete a note |
